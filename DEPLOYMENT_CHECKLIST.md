# Render部署检查清单

## 📋 部署前检查

- [ ] 所有代码已提交到GitHub
- [ ] `requirements.txt` 包含所有依赖
- [ ] `Procfile` 存在且内容正确
- [ ] `.gitignore` 已创建
- [ ] 敏感信息已从代码中移除

## 🚀 部署步骤

- [ ] 在Render创建账户
- [ ] 连接GitHub仓库
- [ ] 创建Web Service
- [ ] 设置环境变量
- [ ] 创建PostgreSQL数据库
- [ ] 添加数据库环境变量
- [ ] 点击Deploy

## ✅ 部署后检查

- [ ] 应用显示"Live"状态
- [ ] 可以访问应用URL
- [ ] 登录页面正常加载
- [ ] 数据库连接正常
- [ ] 可以创建新用户
- [ ] 可以登录
- [ ] 问卷填写功能正常

## 🔧 故障排除

如果部署失败：

1. 检查Logs标签查看错误信息
2. 验证环境变量是否正确
3. 确认数据库是否已创建
4. 检查requirements.txt是否有语法错误
5. 查看Render文档或社区论坛
